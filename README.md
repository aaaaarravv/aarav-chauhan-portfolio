# Aarav Chauhan - Portfolio
### Created as part of an assignment by InAmigos Foundation.
---
A modern, premium React-based portfolio showcasing projects, skills, and experience in Data Science, AI Systems, and Defence Technologies.

## https://aarav-chauhan-portfolio.netlify.app/

## Features

- **Modern Design**: Clean, startup-oriented aesthetic with premium typography
- **Responsive**: Fully responsive across all devices
- **ATS-Friendly**: Structured content optimized for readability
- **Project Showcase**: Detailed project cards with GitHub links
- **Interactive UI**: Smooth animations and hover effects

## Tech Stack

- React 18
- Vite
- CSS3 (Custom Properties)
- Inter Font Family

## Getting Started

### Prerequisites

- Node.js (v16 or higher)
- npm or yarn

### Installation

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

8. **Footer** - Social links and copyright

## Customization

The portfolio uses CSS custom properties for easy theming. Edit the `:root` variables in `src/index.css`:

```css
:root {
  --primary: #0A0E27;
  --secondary: #1A1F3A;
  --accent: #4F46E5;
  --accent-light: #6366F1;
  /* ... */
}
```
